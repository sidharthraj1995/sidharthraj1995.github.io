## **Customization**

If you'd like to explore or modify the site:

- Main content lives in **index.html**
- Styles are in **sass/styles.scss** (compiled to `css/styles.css`)
- Scripts are in **js/scripts.js**
- Images and project assets are stored under **/images**

To rebuild styles or scripts:

```bash
npm install
npm run watch
```

This compiles Sass and minifies JavaScript automatically.
